# noun=conversation — Partition Rationale

Per-prefix rationale for the `conversation` noun:
- `verb=X.md` — verb semantics within conversation, shared across its
  locator forms.
- `locator=X.md` — addressing semantics within conversation, shared
  across the verbs that accept the locator.

## What belongs

- A verb's behavior at the conversation level (cross-locator).
- A locator's addressing semantics (cross-verb within conversation).

## What does NOT belong

- 3-key per-script intersections (`conversation × browse × url` flow) —
  script docstrings.
- Conversation storage layout — `../../chat-as-directory.md`.
- The bare per-noun summary — `../noun=conversation.md`.
- Cross-noun verb-naming rationale — `../verb=*.md` or parent prose.

# Procfs-Style Control Plane (Future)

Learn to implement writable virtual files that trigger actions and readable
virtual files that produce dynamic content. This is the pattern chatfs needs
for `.sync`, `needs_sync/`, `control`, and `status` files.

**Status:** Future, unscheduled milestone. To be tackled when the developer is
ready, after mastering basic FUSE read operations.
